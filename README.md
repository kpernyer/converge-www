# converge.hey.sh

Website for the [Converge](https://github.com/kpernyer/converge-core) multi-agent runtime.

## Development

```bash
bun install
just dev
```

Open http://localhost:5173

## Commands

```
just dev            # Start development server
just build          # Build for production
just check          # Run lint, typecheck, and build
just deploy         # Deploy to Firebase
```

## Stack

- React 19
- Vite
- TypeScript
- CSS Modules
- Firebase Hosting
